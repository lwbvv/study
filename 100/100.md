```text
##오답 저런식으로 스플릿을 하게 되면 문자열 전체가 단 하나의 리스트 요소로 들어간다
## ex) a = "asd"
##     b = a.split() ### b = ['asd']
import io, sys
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf8')

a = str(input()).split()

for char in a:
    print("'"+char+"'\n")


정답
import io, sys
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf8')

a = list(str(input()))

for char in a:
    print("'"+char+"'")
```
